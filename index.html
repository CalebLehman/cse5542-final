Welcome

You have reached the landing page for Caleb Lehman's Real-Time Rendering (CSE 5542) final project. This project uses techniques learned throughout this class to render a scene with 3 famous knots from knot theory; the unknot, the trefoil knot (an exapmle of a torus knot), and a figure-eight knot (an example of a double torus knot). From this page you can navigate to documentation for the project functionality and goals, as well as the project itself.

For Grader

The usage of the program is described on the bottom of the project page. In addition to the required elements of the project, I also implemented bump mapping (normal mapping) in the texture shader. The "Brick," "Scales," and "Organic" textures all have some normal mapping component. I tested my project on Chrome 78 and Firefox 70 using both a local HTTP server and by accessing through a link hosted in the GitHub repository.
